# Skill: GitHub Extractor

## Goal
Build safe, resilient server-side logic to retrieve public GitHub account data.

## Responsibilities
- normalize GitHub URL
- validate profile path
- fetch public data
- parse HTML when needed
- return typed models
- handle partial extraction safely

## Rules
- public data only
- server-side only
- restrict outbound domains
- no brittle hardcoded scraping assumptions when avoidable

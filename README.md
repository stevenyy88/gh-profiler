# GitHub Profiler MVP

A web application that accepts a GitHub user profile URL and produces a deterministic structured summary of the account using public data. 

## Overview
This application provides insights into:
- The account owner's profile and bio
- Predominant languages used based on public repositories
- Star and fork metrics across their public repos
- Account freshness and maintenance signals
- Notable repositories
- Recent contribution volume
